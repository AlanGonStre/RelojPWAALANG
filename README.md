# RelojPWAALANG
Aplicación web progresiva que muestra la hora actual del sistema y la actualiza cada segundo.

## Archivos del proyecto
- index.html
- styles.css
- app.js
- manifest.json
- service-worker.js
- icono.png

## Funcionalidades
- Visualización de la hora actual
- Actualización automática cada segundo
- Registro de service worker
- Instalación como PWA
